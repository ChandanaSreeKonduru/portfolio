                    <div class="tech-discovery-grid">
                        <div class="td-card">
                            <h4>Wireless Protocol Complexity</h4>
                            <p>ESLs use Zigbee mesh networks. Mobile app needed to bridge Zigbee ↔ WiFi ↔ Cloud without store IT changes. Solution: Gateway device API abstraction—app talks HTTP, gateway handles Zigbee.</p>
                        </div>
                        <div class="td-card">
                            <h4>e-ink Latency Reality</h4>
                            <p>Display updates take 2-3 seconds (e-ink refresh). Staff perceived app as "slow" when it was actually display hardware. Solution: Optimistic UI—show "updating" immediately, confirm when display ACKs.</p>
                        </div>
                        <div class="td-card">
                            <h4>NFC/QR Trade-offs</h4>
                            <p>NFC: instant, requires tap, ~80% phone compatibility. QR: universal, requires camera alignment, slower. Solution: Support both, prioritize NFC when available, fallback to QR.</p>
